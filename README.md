## How to Launch

### Prerequisites

- Go 1.21 or higher
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/tiblocko2/payment_system.git
cd payment_system
```

2. Install dependencies:
```bash
go mod download
```

### Running the Program

```bash
go run main.go
```
